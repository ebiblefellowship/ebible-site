Help Installing and Running the Paltalk Messenger

Installing the Paltalk Messenger software

Installing the Paltalk Messenger software will enable you to listen to our Sunday messages and ask questions during the Open Forum and Fellowship Hour times.

Speakers, headphones or headset are needed to listen. During Bible Q&A times, questions can be asked with a microphone or entered by text.

To install the Paltalk Messenger, simply go to the Paltalk Messenger Download web site and follow the instructions for signing up as a new user. As part of that process, you will be instructed to download and install the Paltalk Messenger software.

However, currently, the Paltalk Messenger can only be installed on computers running Microsoft Windows™. Machines running other operating systems such as Apple OS X or Linux are not currently supported although Paltalk now mentions on their web site that a Mac version is, “coming soon.” In the meantime, Mac OS X and Linux users running Intel or AMD processors should have success running under a virtual machine such as one by VMware. However, this requires a licensed copy of Windows™ as well.

Another option for OS X and Linux users that as of this writing doesn’t quite work is the Wine or CodeWeavers CrossOver Mac/Linux Windows compatibility layers.

In any case, one can listen over the corresponding MP3 or Ogg Vorbis streams if there’s one for the associated Paltalk Chat Room.

Running the Paltalk Instant Messenger the First Time

Once you have the Paltalk Instant Messenger installed, start it, create a nickname and login. Click the Chat Rooms tab. Near the bottom of the window, enter “EBible” (without the quotes) into the Search For Chat Rooms search box and click the Search button. Double-click on the desired room name such as EBible Fellowship or EBible Reading Fellowship. Once you’ve entered the room, add the room to the Paltalk Messenger Favorites to make it easier to enter it in the future. For the complete list of rooms maintained by EBible Fellowship, check here.
